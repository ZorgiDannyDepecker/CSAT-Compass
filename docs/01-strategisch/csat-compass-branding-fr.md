## 1. Le nom

**CSAT-Compass** — *Customer Satisfaction Compass*

> *"La boussole de satisfaction client qui oriente chaque pilier ZORGI."*

---

## 2. La métaphore de la boussole

Une boussole indique toujours le chemin — même lorsque la direction est incertaine.
C'est précisément ce que fait CSAT-Compass : il donne une direction aux organisations
qui cherchent des repères dans les retours clients.

### 2.1 Les 4 points cardinaux = les 4 piliers ZORGI

```text
                    ↑ NORD
                    PHARMA
              Pharmacie hospitalière

    ← OUEST                       EST →
    CARE ADMIN                     CARE
    Administration                 Soins infirmiers

                    ↓ SUD
                    ERP4HC
              Systèmes d'entreprise
```

| Point cardinal | Pilier     | Domaine                                    |
|----------------|------------|--------------------------------------------|
| ↑   Nord       | PHARMA     | Pharmacie hospitalière                     |
| → Est          | CARE       | Applications de soins infirmiers           |
| ← Ouest        | CARE ADMIN | Applications administratives de soins     |
| ↓   Sud        | ERP4HC     | Systèmes d'entreprise pour la santé        |

> 🇫🇷 *"Votre boussole de satisfaction pointe dans les 4 directions."*

---

## 3. Pourquoi "Compass" plutôt que "Insights" ou "Analytics"

| Alternative | Problème                                          |
|-------------|---------------------------------------------------|
| `Insights`  | Sec, générique — utilisé par tout le monde        |
| `Analytics` | Trop technique — évoque un outil, pas une direction |
| `Dashboard` | Décrit le quoi, pas le pourquoi                   |
| `Compass`   | ✅ Navigation, direction, repère — une histoire    |

Une boussole **navigue**. Elle aide les organisations "perdues" dans les retours
clients à trouver la bonne direction. C'est précisément la promesse de ce projet.

---

## 4. Analyse de la dénomination

### 4.1 Points forts

- **La métaphore est universelle** — la boussole est culturellement neutre en NL et FR
- **Pas trop marketing** — suffisamment sérieux pour un environnement de développement
- **Pas trop technique** — compréhensible pour le management et les parties prenantes
- **Attrait GitHub/dev-community** — créatif mais professionnel
- **Visuellement fort** — l'emoji 🧭 renforce directement la métaphore

### 4.2 Pérennité

```text
CSAT-Compass        → projet actuel (focus ZORGI PHARMA)
       ↓
ZORGI-Compass       → extension à tous les piliers ZORGI
       ↓
Compass-Analytics   → évolution vers une plateforme d'analyse complète
```

L'**espace de nommage** est libre et le nom évolue logiquement avec la croissance du projet.

---

## 5. Positionnement

> **CSAT-Compass** est la plateforme d'analyse interne de ZORGI qui transforme
> les données mensuelles de satisfaction client en insights exploitables pour
> les quatre piliers. La boussole donne une direction — des données brutes
> de ticketing aux décisions stratégiques.
